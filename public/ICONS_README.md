 ### Конфигурация
 - `favicon.svg` - основной фавикон
 - `manifest.json` - PWA манифест
 - `browserconfig.xml` - для Windows/IE
 - `robots.txt` - для поисковых роботов
 - `sitemap.xml` - карта сайта
 
 ## Проверка
 
 Для проверки PWA используйте:
 1. Chrome DevTools → Lighthouse → Progressive Web App
 2. Chrome DevTools → Application → Manifest
 
 ## Цвета бренда
 - Primary: #06FFBF
 - Background: #1A1A1A
 - Text: #0D0D0D
 
 ## Обновление иконок
 Для перегенерации иконок запустите:
 ```bash
 node generate-icons.js
 ```
